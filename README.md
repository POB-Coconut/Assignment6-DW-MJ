## 과제 구현 목록

### Timer

- 재사용성 고려한 Timer 컴포넌트
- Date 객체를 이용한 한국 / 미국 지역 시간 표기

### Clock

- setInterval 이용한 시계 기능 구현

### Sort

- Quick Sort를 이용한 숫자 정렬.
- option을 이용한 오름차순 / 내림차순 정렬 기능

### Validate

- 정규식을 이용한 형식 체크
